Technical Details

⚙️ How It Works Under the Hood

  • • Client-side only: All processing uses your browser's JavaScript engine
  • • PDF generation: Powered by PDF-Lib, a pure JavaScript PDF library
  • • Image processing: Canvas API reads and resizes PNGs for thumbnails
  • • File handling: FileReader API reads files without uploading
  • • Storage: No files are ever stored - they exist only in memory

📊 Specifications

  • • File support: PNG format only (others will be rejected)
  • • Maximum files: Unlimited (performance depends on your device)
  • • File size: No hard limit - uses browser memory
  • • PDF standards: Generates PDF 1.7 compatible files
  • • Browser support: Chrome, Firefox, Edge, Safari (modern versions)

Frequently Asked Questions

Is this really private? ▼

Yes! Your files never leave your device. We don't have servers, don't collect data, and don't track anything. All conversion happens locally in your browser.

What's the maximum file size? ▼

There's no hard limit - it depends on your device's available memory. Very large files (100MB+) might work slowly on older devices, but there's no artificial restriction.

Can I convert other image formats? ▼

Currently we support PNG only. This ensures optimal quality and compatibility. For JPEG or other formats, you can convert them to PNG first using any image editor.

Is there a limit on number of files? ▼

No limit! You can add as many as you want. The interface handles reordering and combining efficiently.

Can I use it offline? ▼

Once the page is loaded, yes! All libraries are loaded with the page. After that, you can disconnect from the internet and still convert files.
